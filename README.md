# Gradient Interference for Continual Learning

This repository contains all Colab files for the MIE424 course project, including:
- Data processing for CIFAR-10, Office-Home, and DomainNet.
- Implementation of gradient interference methods:
  - Orthogonal gradient descent (OGD)
  - Gradient episodic memory (GEM)
  - Gradient surgery (PCGrad)
- Experiments comparing the effectiveness of the methods using different datasets.
